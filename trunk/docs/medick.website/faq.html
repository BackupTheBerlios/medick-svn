medck F.A.Q.

  • Q. What is medick?
    A. medick is an open-source web framework.
  • Q. Where did the name "medick" come from?
    A. It's from Black Medick - Medicago lupulina L.
  • Q. What are medick requirements?
    A. medick needs php (version 5.1.2 or higher), a web server (Apache with mod_php) and optionally a database server.
  • Q. What license is medick released under?
    A. I choose to go open source and use a BSD style licence for medick.
  • Q. Is medick suitable for use yet?
    A. medick it's still a alpha stage project so it's suitable for use as long as you understand that it may undergo large backwards incompatible changes.
  • Q. Is medick being worked on?
    A. Yes, but sporadically. It's mainly an experiment and a playground, though it does seem to be a very good one. If you want to see it be furthered, feel free to write code in it, give feedback, and contribute.
